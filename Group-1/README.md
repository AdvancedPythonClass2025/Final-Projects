 ## Project Overview
 - **Description:** This project is an MP3 player that allows users to play and manage audio files.
 - **Main Features:**
   1. Play MP3 files
   2. Manage playlists
   3. Graphical User Interface (GUI) using custom Tkinter
 - **Tech Stack:**
   - Python 3.8+
   - Custom Tkinter (for GUI)
